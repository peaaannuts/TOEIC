# TOEIC 600 学習アプリ

TOEIC L&R で600点を目指す1ヶ月集中学習アプリです。
サーバー不要・インストール不要のWebアプリ(PWA)として動きます。

## 機能

- **ホーム**: 試験日までのカウントダウン、今日のノルマ進捗、連続学習日数、習得単語数
- **単語**: 600点レベル頻出の約310語を暗記カードで学習。
  「覚えた/まだ」の結果に応じて復習間隔が自動調整されます(1日→3日→7日→14日)。
  発音記号つき・カード表示時に自動読み上げ(🔊で聞き直し、例文の読み上げも可)
- **文法**: 本番Part 5と同じ短文穴埋め4択クイズ(300問、和訳・日本語解説つき)。
  単語と同じ間隔反復方式で、正解すると次の出題が1日→3日→7日→14日後ろへ延び、
  間違えるとリセットされます。復習期日が来た問題から優先的に出題されます
- **リスニング**:
  - **Part 1形式(写真描写)16問**: アプリ内蔵のイラストを見て、読み上げられる
    4つの説明文から最も合うものを選びます(1セット6問、本番と同じ問題数)
  - **Part 2形式(応答問題)60問**: 質問と3つの応答が読み上げられ、正しい応答を選びます
  - どちらも端末内蔵の音声合成を使用。解答後にスクリプト・和訳・解説を表示。
    音声が使えない環境では英文を読んで解く形式に自動で切り替わります
  - 単語・文法と同じ間隔反復方式(正解で次の出題が1日→3日→7日→14日後ろへ、
    間違えるとリセット)。復習期日が来た問題から優先的に出題されます
- **読解**: 本番Part 7と同じ、Eメール・お知らせ・広告・記事・求人などの文書と、
  ダブルパッセージ(2文書を照合して解く問題)を収録(10文書セット・28問、全設問に和訳・解説つき)。
  文書を読みながら1問ずつ解答し、**読解タイマー**で「1分あたり何問解けたか」を表示するので、
  本番Part 7(約54問を約55分)の時間配分の感覚が養えます。
  文書セット単位の間隔反復方式(セット内全問正解でレベルUP、1問でも間違えるとリセット)
- **記録**: 直近2週間の学習量グラフ、タイプ別正答率、苦手単語リスト
- **モチベーション**: XP&レベル&称号、連続正解コンボ(正解音のピッチが上がる)、
  デイリーチャレンジ(毎日3つ・達成でボーナスXP)、実績バッジ15種、
  タイムアタックモード(文法・1問20秒+スピードボーナス)、レベルアップ/パーフェクト時の紙吹雪演出

データはブラウザの localStorage に保存されます(端末ごと)。

## パソコンで試す

このフォルダで簡易サーバーを起動してブラウザで開きます:

```
npx serve .
```

(`index.html` をダブルクリックして直接開いても動作しますが、その場合オフラインキャッシュは無効です)

## スマートフォンで使う

スマホで使うにはインターネット上に公開するのが簡単です。無料の方法:

1. **Netlify Drop**(アカウント不要ですぐ試せる)
   https://app.netlify.com/drop にこのフォルダをドラッグ&ドロップ → 発行されたURLをスマホで開く
2. **GitHub Pages**
   GitHubリポジトリにこのフォルダをプッシュ → Settings → Pages で公開

スマホのブラウザで開いたら、**「ホーム画面に追加」** をすると
アイコン付きの普通のアプリのように全画面で使えます(オフラインでも動作)。

- iPhone (Safari): 共有ボタン → 「ホーム画面に追加」
- Android (Chrome): メニュー → 「ホーム画面に追加」/「アプリをインストール」

## 600点への使い方の目安(1ヶ月プラン)

- 単語: 1日20枚(デフォルトのノルマ)。新しい単語と復習が自動で混ざります
- 文法: 1日10問(1セット)。**間違えた問題の解説を読むのが一番大事**
- リスニング: 1日10問(1セット)。間違えた問題はスクリプトを見ながらもう一度🔊で聞き直すと効果的
- 週末: 記録タブの「苦手な単語」を見直し
- 試験1週間前からは、市販の公式問題集で模試を1回解いて時間配分に慣れるのがおすすめです

## ファイル構成

| ファイル | 役割 |
|---|---|
| `index.html` | 画面構造 |
| `style.css` | デザイン(モバイルファースト) |
| `data.js` | 単語・問題データ |
| `app.js` | ロジックとデータ保存 |
| `manifest.webmanifest` / `icon.svg` / `sw.js` | PWA対応(ホーム画面追加・オフライン動作) |
