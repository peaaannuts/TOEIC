# TOEIC 600 学習アプリ

TOEIC L&R で600点を目指す1ヶ月集中学習アプリです。
サーバー不要・インストール不要のWebアプリ(PWA)として動きます。

## 機能

- **ホーム**: 試験日までのカウントダウン、今日のノルマ進捗、連続学習日数、習得単語数
- **単語**: 600点レベル頻出の約310語を暗記カードで学習。
  「覚えた/まだ」の結果に応じて復習間隔が自動調整されます(1日→3日→7日→14日)。
  発音記号つき・カード表示時に自動読み上げ(🔊で聞き直し、例文の読み上げも可)
- **文法**: 本番Part 5と同じ短文穴埋め4択クイズ(300問、和訳・日本語解説つき)。
  単語と同じ間隔反復方式で、正解すると次の出題が1日→3日→7日→14日後ろへ延び、
  間違えるとリセットされます。復習期日が来た問題から優先的に出題されます
- **リスニング**:
  - **Part 1形式(写真描写)16問**: アプリ内蔵のイラストを見て、読み上げられる
    4つの説明文から最も合うものを選びます(1セット6問、本番と同じ問題数)
  - **Part 2形式(応答問題)74問**: 質問と3つの応答が読み上げられ、正しい応答を選びます。
    「まだ決まっていない」「〜に聞いて」のように直接答えない**間接応答**の問題も含み、本番の難所に対応
  - **Part 3形式(会話問題)8会話・24問**: 2〜3人の会話を聞いて、内容に関する3つの設問に答えます。
    話者ごとに声を変えて再生します(会話・トークは音声のみ、設問と選択肢は文字で表示)
  - **Part 4形式(説明文問題)6トーク・18問**: 留守番電話・館内アナウンス・広告・ツアー案内・
    交通情報・研修の冒頭など、1人による短いトークを聞いて3つの設問に答えます
  - いずれも端末内蔵の音声合成を使用。解答後にスクリプト・和訳・解説を表示。
    音声が使えない環境では英文を読んで解く形式に自動で切り替わります
  - Part 1・2は問題単位、Part 3・4は会話/トーク単位の間隔反復方式(正解で次の出題が
    1日→3日→7日→14日後ろへ、間違えるとリセット)。復習期日が来たものから優先的に出題されます
- **読解**: 本番のReadingセクションを2種類収録。
  - **Part 6形式(長文穴埋め)6長文・24問**: 長文中の4つの空所を、文法・語彙・つなぎ言葉や
    「文挿入」問題で埋めます。今解いている空所が本文中でハイライトされます
  - **Part 7形式(読解)13文書セット・48問**: Eメール・お知らせ・広告・記事・求人・オンラインレビュー・
    日程表などの文書と、ダブルパッセージ(2文書を照合して解く問題)。文書を読みながら1問ずつ解答します。
    本番頻出の**語彙言い換え問題**(下線語に最も近い意味を選ぶ)や**推測問題**も収録
  - どちらも全設問に和訳・解説つき。**読解タイマー**で「1分あたり何問解けたか」を表示するので、
    本番Part 7(約54問を約55分)の時間配分の感覚が養えます
  - 文書セット単位の間隔反復方式(セット内全問正解でレベルUP、1問でも間違えるとリセット)
- **推定スコア**: ホーム画面に、これまでの演習成績(正答率)から算出したTOEIC L&Rの目安スコアを表示。
  Listening(Part 1・2)/Reading(文法・読解・単語)の内訳と、目標スコアまでの残りや
  「次はどちらを重点的にやるべきか」の診断も出ます(目標スコアは設定で変更可)
- **記録**: 直近2週間の学習量グラフ、タイプ別正答率、苦手単語リスト
- **モチベーション**: XP&レベル&称号、連続正解コンボ(正解音のピッチが上がる)、
  デイリーチャレンジ(毎日3つ・達成でボーナスXP)、実績バッジ15種、
  タイムアタックモード(文法・1問20秒+スピードボーナス)、レベルアップ/パーフェクト時の紙吹雪演出
- **継続の仕組み(ストリーク)**: ホーム最上部に🔥連続学習日数を大きく表示。今日まだ学習していない日は
  炎がグレーになり、学習すると点灯します。3/7/14/30/50/100日の節目でボーナス💎ジェムがもらえます
  - **❄️お守り(ストリークフリーズ)**: 1日休んでも連続記録が途切れないお守り。空いた日を自動で
    埋めます(最大2個)。7日ごとの節目で1個もらえるほか、ショップで💎ジェムと交換できます
  - **💎ジェムと🎁宝箱**: デイリーチャレンジを達成すると宝箱が開き、ランダムな量のジェムがもらえます。
    貯めたジェムは炎カードをタップして開く**ショップ**でお守りと交換できます
  - **🗓️ 週間クエスト**: 週替わりで3つの目標(例:今週リスニングを80問/5日学習する)。1日では
    終わらない目標なので、毎日コツコツ続ける動機になります。達成でXPとジェムがもらえます
  - **🏁 先週の自分との対戦(ゴーストリーグ)**: 今週の獲得XPで「先週の自分」を超えられるか競います。
    ホームの「今週」カードに対戦バーが出て、週明けに勝敗を演出(勝つとボーナスジェム)
  - **🏆 月間バッジ**: デイリー・週間クエストをその月に一定数こなすと、月ごとの限定バッジを獲得。
    記録タブに額縁コレクションとして残ります

データはブラウザの localStorage に保存されます(端末ごと)。

## パソコンで試す

このフォルダで簡易サーバーを起動してブラウザで開きます:

```
npx serve .
```

(`index.html` をダブルクリックして直接開いても動作しますが、その場合オフラインキャッシュは無効です)

## スマートフォンで使う(GitHub Pagesで自動公開)

このリポジトリには GitHub Pages への**自動デプロイ**が設定されています
(`.github/workflows/deploy-pages.yml`)。`main` ブランチの `toeic/` 配下が
更新されるたびに、`toeic/` フォルダがサイトのルートとして自動で公開されます。

- 公開URL: **https://peaaannuts.github.io/TOEIC/**
- 更新方法: `main` または開発ブランチ(`claude/toeic-app-dev-st2txs`)にプッシュするだけ(数分でPagesに反映)。
  **mergeしなくても、開発ブランチへのプッシュだけで公開URLに反映されます。**

### 初回だけ必要な設定

リポジトリの **Settings → Pages → Build and deployment → Source** を
**「GitHub Actions」** に設定してください(初回のワークフロー実行で自動的に
有効化される場合もあります)。以降はプッシュするだけで更新されます。

デプロイの状況は GitHub の **Actions** タブで確認できます。

スマホのブラウザで公開URLを開いたら、**「ホーム画面に追加」** をすると
アイコン付きの普通のアプリのように全画面で使えます(オフラインでも動作)。

> 反映されないときは、アプリを一度閉じてから2回開き直すと新しいバージョンに更新されます
> (Service Workerのキャッシュ更新のため)。

### 学習記録について(消えない・引っ越せる)

学習記録は各自の端末のブラウザ内(localStorage)に、**その URL ごと**に保存されます。

- **アプリを更新しても記録は消えません**。同じ URL で使い続ける限り、更新後もそのまま残ります。
- ただし記録は URL に紐づくため、**別の URL(以前使っていた Netlify のアドレスなど)の記録は、
  新しい github.io の URL には自動では引き継がれません**。

引っ越しや機種変更に備えて、**設定(⚙️)→「学習記録のバックアップ」** から記録を書き出し/復元できます。

- 古い URL でアプリを開く →「📤 書き出す」でバックアップファイルを保存
- 新しい URL(https://peaaannuts.github.io/TOEIC/ )で開く →「📥 ファイルから復元」でそのファイルを読み込む
  (ファイルが扱いにくいスマホでは「テキストを貼り付けて復元」も使えます)

#### 別の手段(任意)

GitHubを使わずすぐ試したいときは **Netlify Drop**(https://app.netlify.com/drop )に
`toeic` フォルダをドラッグ&ドロップしても公開できます。

- iPhone (Safari): 共有ボタン → 「ホーム画面に追加」
- Android (Chrome): メニュー → 「ホーム画面に追加」/「アプリをインストール」

## 600点への使い方の目安(1ヶ月プラン)

- 単語: 1日20枚(デフォルトのノルマ)。新しい単語と復習が自動で混ざります
- 文法: 1日10問(1セット)。**間違えた問題の解説を読むのが一番大事**
- リスニング: 1日10問(1セット)。間違えた問題はスクリプトを見ながらもう一度🔊で聞き直すと効果的
- 週末: 記録タブの「苦手な単語」を見直し
- 試験1週間前からは、市販の公式問題集で模試を1回解いて時間配分に慣れるのがおすすめです
